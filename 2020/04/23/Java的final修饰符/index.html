Java的final修饰符

2020-04-23- 2020-05-13
-
学习历程 > Java-Java

final关键字：

  • ​ 修饰类：不能被继承

  • ​ 修饰方法：不能被重写

  • ​ 修饰变量：变量不能被重新赋值（只能初始化一次）

  • ​ 修饰引用类型：地址不可变（例如：新建一个对象，地址不能改变，但对象内成员变量的值可以改变）

Tip:被final修饰的成员变量必须赋初值，被final修饰的局部变量只能初始化一次但不一定要赋初值
